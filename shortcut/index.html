Mode Shorcut

G (+X, +Y, +Z) (+有理数) オブジェクトの移動(Grab:掴む) 例) GX -> X座標の方向にだけ動かせる 例) GZ10 -> Z座標の方向に10m動かす
G (+XX, +YY, +ZZ) (+有理数) 動かす軸をローカル軸に変更する
Alt + G オブジェクトモードで動かしたGの値をリセットする
S (+X, +Y, +Z) (+有理数) オブジェクトの拡大/縮小(Scal:大きさ) 例) SX -> Y座標の方向にだけ大きさを変えられる 例) SY10 -> Y座標の方向に10倍大きくする
S (+XX, +YY, +ZZ) (+有理数) 拡大/縮小軸をローカル軸に変更する
Alt + S オブジェクトモードで動かしたSの値をリセットする
R (+X, +Y, +Z) (+有理数) オブジェクトの回転(Rotate:回転) 例) RZ -> Z座標の方向にだけ回転できる 例) RX10 -> X座標の方向に10度回転させる
R (+XX, +YY, +ZZ) (+有理数) 回転軸をローカル軸に変更する
Alt + R オブジェクトモードで動かしたRの値をリセットする
Shift + D オブジェクトの複製
Alt + D オブジェクトのリンク複製 編集モードで動かすとリンク複製したオブジェクトも同じ挙動をとる
H オブジェクトの非表示
Alt + H オブジェクトの再表示
W カーソル種類切り替え
Shift + C 3Dカーソル位置リセット
X 削除
Q お気に入りメニューの表示
A 全選択
Alt + A 全選択解除
Shift + Z 透過表示切り替え
Shift + S パイメニュー表示
Ctrl + Alt + Q 画面4分割
