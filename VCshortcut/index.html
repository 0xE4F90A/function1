VisualStudio

Custom Shortcut

Shortcut

Ctrl + / // カーソルがある行のコメント切り替え
Ctrl + D // 同一単語の複数選択
Alt + ↑ or ↓ // カーソルがある行を、上もしくは下に移動
Shift + Alt + ↑ or ↓ // カーソルがある行を、上もしくは下に複製
Ctrl + Alt + クリック // クリックした個所にカーソルを複製 既にカーソルがある場合、カーソルを削除
Ctrl + Alt + ↑ or ↓ // カーソルがある行から、上もしくは下にカーソルを複製
Ctrl + クリック // 変数もしくは関数の定義に移動
Shift + Alt + ドラッグ // カーソルから縦軸変更せず複数行選択可能
F5 // デバッグの実行
Ctrl + F // 検索画面の表示
Shift + Alt + F // コードの自動整形

Others Shortcut

Ctrl + S // 保存
Ctrl + C // コピー /* 選択していない場合、カーソルのある行をコピーできる */
Ctrl + X // 切り取り /* 選択していない場合、カーソルのある行をカットできる 使わない行を一括で削除可能 */
Ctrl + V // 貼り付け
Windows + V // クリップボードに溜められている過去のコピーを閲覧・ペースト可能
Ctrl + Z // 直前の操作を取り消す
Ctrl + Y // 直前の操作を繰り返す
Shift + Alt // 右下の A/あ ⇔ ENG 切り替え /* 日本語が入力できなくなる原因の9割がこいつのせい */
F2 // 名前を変更
Ctrl + Shift + Esc // タスクマネージャーを起動
Ctrl + Alt + Delete // セキュア・アテンション画面を起動 /* 操作不能時に試してみるといいかも */
Windows + L // 画面をロック /* 離席時に使用しよう */
